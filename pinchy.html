Description

Internet of things – seamless change of monitors depending on user context. The user gets a prompt message offering a switch of device and responds with a finger movement.

Method

The communication between the devices is done by the use of PubNub, a service for internet of things. An Arduino is used for monitoring sensors that determine the context, while a web application is used to display the UI.
